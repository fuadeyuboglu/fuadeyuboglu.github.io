DeLorean: the time machine!

The DeLorean time machine is a time travel device made by retrofitting a DMC DeLorean vehicle with a flux capacitor, which allows for time travel when it hits 88 miles per hour.

  • First appearence: Back to the Future 1985
  • Created by: Robert Zemeckis and Bob Gale
  • Type: Time machine
